Приложение "Мои митапы" позволяет пользователю :

- подписаться на определенный митап, который он хочет посетить ;

- создать новый митап ;

Вообще, митап - эта такая карточка где хранятся данные о предстоящем событии: его описание, дата, что на нем будет и т.д. Для того чтобы попасть на митап, тебе нужно пройти регистрацию пользователя . Регистрация доступна по ссылке ниже
